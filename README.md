# 東松山バス あと何分？ (PWAアプリ)

東松山駅〔東口〕〜パークタウン五領線のバスが「あと何分何秒で来るか」を開いた瞬間に確認できる、シンプルで高速なPWA（Progressive Web App）アプリです。

## ✨ 特徴
- **開いた瞬間わかる**: 余計な操作なしで、次のバスまでの残り時間が秒単位でカウントダウンされます。
- **完全オフライン対応**: 時刻表データをアプリ内に内蔵しているため、電波の悪い場所や速度制限下でも爆速で起動します。
- **行き・帰り自動判定**: 午前（12時前）は「行き（駅方面）」、午後（12時以降）は「帰り（パーク方面）」のタブが最初から選択された状態で起動します。
- **状況がすぐわかる文字色**: 残り時間に応じて、文字色が自動で変化します（通常: 黒 / 10分以内: 緑 / 3分以内: 赤）。

## 📁 フォルダ構成
GitHubリポジトリには、以下のファイルがすべて同じ階層（ルートディレクトリ）に並んでいる必要があります。

```text
├── index.html        (画面の骨組み)
├── style.css         (デザイン・改行防止設定)
├── app.js            (時刻表データとカウントダウン処理)
├── manifest.json     (PWAの設定ファイル)
├── service-worker.js (オフライン起動のためのキャッシュ制御)
├── icon-192.png      (ホーム画面用アイコン 192x192)
└── icon-512.png      (ホーム画面用アイコン 512x512)
